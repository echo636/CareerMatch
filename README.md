一、项目简介
本项目是一个基于 AI 与数据结构化分析的简历岗位智能匹配系统。系统通过对用户简历与岗位信息进行结构化处理与语义理解，结合向量检索与规则打分算法，实现岗位推荐与匹配度评估。
系统目标：
- 提升求职者岗位匹配效率
- 提供可解释的匹配评分
- 输出个性化能力差距（Gap）分析

三、系统核心功能
1. 简历解析与结构化
- 支持 PDF / Word 简历上传
- 提取文本信息
- 使用 LLM 解析为结构化 JSON：
  - 基本信息
  - 技能标签（Tag）
  - 项目经历
  - 工作经验
  - 薪资期望

---
2. 岗位数据处理
- 后台支持批量导入岗位数据
- 原始 JD（非结构化） → LLM 提取 → 标准化 JSON
- 岗位信息拆分为：
  - 技能要求（标签化）
  - 项目经验要求
  - 薪资区间
  - 硬性条件

---
3. 向量化与语义检索
- 简历与岗位均转换为向量表示
- 存入向量数据库
- 使用 KNN（近邻搜索）进行语义召回

---
4. 多阶段匹配算法
第一阶段：粗排（召回）
- 基于向量相似度
- 从岗位库中召回 Top N（如 Top 50）
第二阶段：过滤
- 剔除异常或明显不匹配数据
  - 薪资差距过大
  - 硬性条件不满足
第三阶段：精排（核心）
综合评分模型：
MatchScore = α * 向量相似度
           + β * 技能匹配度
           + γ * 项目匹配度
           + δ * 薪资匹配度
评分规则：
- 技能匹配：按 Tag 重合计分
- 项目匹配：关键经验匹配加权
- 每匹配一项增加权重分
最终输出：
- 按匹配分数排序的岗位列表

---
5. 匹配结果展示
- Top N 匹配岗位列表
- 匹配分数（可解释）
- 岗位详情展示

---
6. Gap 分析（AI亮点）
- 从高匹配岗位中提取对照组
- 分析：
  - 技能差距
  - 薪资差距
  - 经验缺口
输出：
- 结构化 Gap 报告
- 前端可视化（雷达图）

---
7. 可扩展能力（未来）
- 职业成长路径推荐
- 学习路线规划
- 项目推荐

五、核心业务流程
[图片]
1. 岗位数据入库流程
1. 收集原始 JD
2. 批量导入
3. LLM 提取结构化字段
4. 生成标准 JD JSON
5. 向量化
6. 存入：
  - 向量库（语义检索）
  - PostgreSQL（结构化数据）

---
2. 简历处理流程
1. 用户上传简历
2. 文本解析
3. LLM 提取能力信息
4. 生成标准 CV JSON
5. 向量化
6. 存入向量库

---
3. 匹配流程
1. 向量检索（Top N）
2. 数据过滤
3. 精排打分
4. 返回匹配结果列表

---
4. Gap 分析流程
1. 提取 Top 岗位作为对照组
2. 构建对比数据
3. LLM 深度分析
4. 输出 Gap 报告
5. 前端可视化展示

六、项目总结
该系统结合：
- AI（LLM + Embedding）
- 信息检索（向量搜索）
- 规则算法（精排评分）
实现了从数据处理 → 语义理解 → 智能匹配 → 差距分析的一体化闭环。
